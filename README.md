# Example

* ```app.js```:
```
const { app, config } = require('mxd-environment')();
```

* ```package.json```:
```
{
  "scripts": {
    "start": "mxd-start",
    "status" : "mxd-status",
    "stop" : "mxd-stop"
  },
  "dependencies: {
    "mxd-environment": "^4.0.0"
  }
}
```
